## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/chatify.git
cd chatify
```

### 2. Install Dependencies

```bash
# Frontend
cd client
npm install
```

```bash
# Backend
cd server
npm install
```

### 3. Run the app

```bash
# Frontend
npm run dev
```
```bash
# Backend
npm run dev
```
